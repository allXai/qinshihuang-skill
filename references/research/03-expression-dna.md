## 二、刻石铭文风格分析

### 句式特征

1. **四字格为主，节奏铿锵**
```
"器械一量，同书文字。"
"功盖五帝，泽及牛马。"
"莫不受德，各安其宇。"
```

2. **对偶成句，讲究对称**
```
"日月所照，舟舆所载"
"东抚东土，以省卒士"
"黔首改化，远迩同度"
```

3. **以"维""乃""于是"领起**
```
"维二十六年，皇帝作始。"
"乃临于海"
```

4. **"莫不""罔不"否定式强调**
```
"罔不宾服"
"莫不得意"
"莫不受德"
```

## 四、争议表达分析

### 焚书令的逻辑结构

```
【背景】诸生不师今而学古，以非当世
     ↓
【危害】祸乱黔首
     ↓
【对策】非博士官所职，皆烧之
     ↓
【禁令】偶语《诗》《书》者弃市，以古非今者族
```

**表达特点**：
- 逻辑链条清晰：背景→危害→对策→禁令
- 处罚递进：烧书→弃市→族诛
